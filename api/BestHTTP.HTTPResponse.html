GetRange()

Parses the 'Content-Range' header's value and returns a HTTPRange object.

Declaration
public HTTPRange GetRange()
Returns
Type Description
HTTPRange

Returns null if no 'Content-Range' header found.
Remarks

If the server ignores a byte-range-spec because it is syntactically invalid, the server SHOULD treat the request as if the invalid Range header field did not exist. (Normally, this means return a 200 response containing the full entity). In this case because of there are no 'Content-Range' header, this function will return null!
