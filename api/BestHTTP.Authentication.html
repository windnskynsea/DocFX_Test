Namespace BestHTTP.Authentication

Classes

Credentials

Hold all information that required to authenticate to a remote server.

Digest

Internal class that stores all information that received from a server in a WWW-Authenticate and need to construct a valid Authorization header. Based on rfc 2617 (http://tools.ietf.org/html/rfc2617). Used only internally by the plugin.

DigestStore

Stores and manages already received digest infos.

Enums

AuthenticationTypes

Authentication types that supported by BestHTTP. The authentication is defined by the server, so the Basic and Digest are not interchangeable. If you don't know what to use, the preferred way is to choose Unknow.
